HR Hospital

Modern hospital management for Odoo 17

A fully-featured app that helps you track doctors, interns, patients, visits, diagnoses and medical specialties - all with granular security rules and business logic covered by automated tests.

Key Features

  • Doctors & mentors with specialties and intern supervision
  • Patient registry with emergency contacts and age calculation
  • Visit planning, calendar view & duplication prevention
  • Diagnoses with mentor approval workflow for interns
  • Security groups (patient, intern, doctor, manager, admin) & record-rules
  • Printable PDF doctor report (QWeb)
  • Two handy wizards:
    • Diagnosis Report (multi-criteria analytics)
    • Bulk Patient-Doctor Update
  • 100 % demo data, YAML-free XML fixtures
  • Extensive unit tests with coverage
  • Ukrainian translation included (uk_UA.po)

Quick Start

# install requirements, then:
odoo --load=web,hr_hospital -d mydb --init hr_hospital --dev all

Log in as "h_admin / demo" to explore the module with pre-loaded demo records.
